# SOP: 프레스 설비 기동 절차 (Startup Procedure)

## 1. 목적

프레스 설비를 안전하고 일관된 방법으로 기동하기 위한 표준 작업 절차를 정의한다.

## 2. 적용 범위

- 프레스-1호기 ~ N호기
- 설비 기동 및 시험 운전 단계

## 3. 기동 전 점검

1. 비상 정지(E-Stop) 스위치가 해제되어 있는지 확인
2. 보호 커버, 인터록 장치가 모두 정상 상태인지 확인
3. 윤활유, 유압유, 냉각수 등 레벨 확인
4. 주변 작업 공간 정리, 이물 및 장애물 제거
5. 작업자 보호구(PPE) 착용 확인

## 4. 기동 절차

1. 메인 전원 ON
2. 제어반(Control Panel) 전원 스위치 ON
3. HMI 또는 패널에서 “Startup” 모드 선택
4. 초기 자가 진단(Self-check) 완료 여부 확인
5. 이상 알람이 없을 경우, 저속(Jog) 모드로 3~5 cycle 시험 운전
6. 진동, 소음, 전류 값이 정상 범위 내에 있는지 확인
7. 정상 확인 후, 자동 운전(Automatic) 모드로 전환

## 5. 기록

- 기동 시간, 담당자, 특이사항을 운전 일지에 기록
- 초기 이상 징후 발견 시 Trouble Shooting 절차에 따라 조치

# SOP: 프레스 설비 비상 정지 절차 (Emergency Stop)

## 1. 목적

프레스 설비에서 긴급 상황 발생 시, 인명 및 장비 피해를 최소화하기 위한 비상 정지 절차를 정의한다.

## 2. 비상 상황 예시

- 작업자 신체가 위험 영역에 진입한 경우
- 이상 소음, 연기, 화재 징후 발생
- 진동/전류 값이 급격하게 상승하는 비정상 상태
- 구조물 파손, 부품 비산 등 위험 상황

## 3. 비상 정지 절차

1. 즉시 비상 정지(E-Stop) 버튼을 누른다.
2. 가능하다면 주변 작업자에게 큰 소리로 비상 상황을 알린다.
3. 메인 전원을 OFF로 전환한다.
4. 동작부 완전 정지 여부 확인 후, 설비 주변 접근.

## 4. 사후 조치

1. 설비 상태 육안 점검 (누유, 파손, 연기 등)
2. 전기/유압 에너지원 완전 차단(Lock-out/Tag-out)
3. 관리자 및 안전 담당자에게 즉시 보고
4. 상황 기록(시간, 원인 추정, 목격자 등)
5. 원인 분석 및 재발 방지 대책 수립 전까지 임의 재가동 금지

# SOP: 프레스 메인 베어링 교체 절차

## 1. 목적

프레스 설비 메인 베어링을 안전하고 표준화된 방법으로 교체하여 고장 및 사고를 예방한다.

## 2. 준비물

- 신규 베어링 (규격: 제조사 스펙 참조)
- 토크 렌치, 풀러, 해머, 드라이버 세트
- 윤활유/그리스
- 청소용 솔벤트 및 천
- 온도 센서(필요 시)
- 개인 보호구(PPE)

## 3. 작업 전 준비

1. 설비 정지 및 메인 전원 OFF
2. Lock-out/Tag-out 절차 수행
3. 베어링 주변 온도 확인(고온 시 충분히 냉각 후 작업)
4. 작업구역 안전 표지 및 접근 통제

## 4. 베어링 탈거

1. 보호 커버 및 관련 부품 제거
2. 샤프트 위치 표시(재조립 기준점)
3. 풀러를 이용해 베어링을 천천히 분리
4. 샤프트 및 하우징 손상 여부 확인
5. 탈거된 베어링 상태 사진 및 기록(균열, 변색 등)

## 5. 베어링 장착

1. 샤프트 및 하우징 청소, 이물 제거
2. 규정된 방향으로 베어링 삽입
3. 규정 토크에 맞춰 체결(제조사 매뉴얼 참조)
4. 윤활 그리스 도포
5. 보호 커버 재장착 및 체결 토크 확인

## 6. 시운전 및 검증

1. 메인 전원 ON, 저속 모드에서 시험 운전
2. 진동/소음/온도 추세 모니터링(최소 30분)
3. AI0_Vibration, AI1_Vibration 값이 정상 범위(2~8 mm/s)에 있는지 확인
4. 이상 없을 경우 자동 운전 모드 전환 허용

## 7. 기록

- 교체 일시, 담당자, 사용 베어링 규격 기록
- 교체 전후 진동/전류 값 비교 기록
