# 프레스 설비 Trouble Shooting Guide

본 문서는 프레스 설비의 진동/전류 이상 발생 시, 엔지니어가 따라야 할 대표적인 진단 절차를 정리한 것이다.

---

## 1. 고진동 이상 (AI0/AI1 > 12 mm/s)

### 1.1 증상 정의

- 메인 베어링 또는 프레임 진동이 12 mm/s 이상
- 소음 증가, 프레임 떨림, 온도 상승 동반 가능

### 1.2 1차 점검 순서

1. 운전 상태 확인  
   - 최근 공정 조건(하중, Stroke 속도 등) 변경 여부 확인
2. 육안 점검  
   - 누유, 균열, 변형, 풀림 볼트 여부 확인
3. 베어링 상태 점검  
   - 온도, 소음, 윤활 상태 확인
4. 축 정렬 확인  
   - 얼라인먼트 게이지 또는 레이저 정렬 장비 사용
5. 기초/프레임 체결 상태  
   - 앵커 볼트 및 구조물 체결 토크 측정

### 1.3 원인별 조치

- 베어링 마모/파손  
  → 베어링 교체, 윤활 조건 재설정
- 축 정렬 불량  
  → 얼라인먼트 보정, 재조립
- 언밸런스  
  → 회전체 밸런싱 작업, 이물질 제거
- 볼트 풀림  
  → 규정 토크 재체결, 풀림 방지 장치 추가

---

## 2. 전류 이상 (AI2_Current > 50 A 또는 < 25 A)

### 2.1 고전류(Over-current)

**가능한 원인**

- 공정 과부하(재료 경도, 두께 증가)
- 실린더/슬라이드 간섭, 재료 끼임
- 모터 혹은 인버터 이상

**점검 순서**

1. 공정 조건 확인 (재료 변경 여부, 금형 변경 등)
2. 슬라이드/테이블 스트로크 중 걸림 여부 확인
3. 유압/기계식 클러치 및 구동계 마찰 상태 확인
4. 인버터 알람 또는 경고 코드 확인

---

### 2.2 저전류(Under-current)

**가능한 원인**

- 무부하 운전(부하 전달 불량)
- 클러치/커플러 슬립
- 기계적 부하 미작동

**점검 순서**

1. 실제 하중이 걸리고 있는지 확인
2. 커플러/클러치 연결부 마모 또는 슬립 여부 점검
3. 공정 자체가 우회되거나 동작하지 않는지 확인

---

## 3. 복합 이상 (진동 + 전류 동시 이상)

- 고진동 + 고전류 동시:
  - 심각한 기계적 구속, 베어링 파손, 심각한 언밸런스 가능성
  - → 즉시 정지 후 상세 점검 필수
- 고진동 + 저전류:
  - 부하 전달 경로 일부 손상 가능성
- 정상 진동 + 고전류:
  - 공정/유압/전기 계통 이상 우선 확인

---

## 4. 일반 가이드

1. 이상 탐지 후, 최소 10~30분간 트렌드 그래프 확인
2. 센서 자체 이상(배선, 접촉 불량, 교정 상태)을 항상 1회는 의심할 것
3. 동일 설비의 과거 이력과 비교하여 패턴 유사 여부 확인
4. 조치 내역은 반드시 8D Report 또는 정비 이력 DB에 기록
